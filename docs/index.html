Stage325: Verifiable Audit Report Export

Do not just export reports. Export verifiable proof packages.

Stage325 turns public verification infrastructure into a submit-ready audit report package in JSON, HTML, and PDF.

Problem

A normal PDF report can be read, but it cannot prove that the underlying audit history has not been changed.

Solution

Stage325 exports the report together with signed history, public key, and reproducible verification instructions.

Exported Report Files

  • report.json
  • report.html
  • report.pdf
  • verify.txt

Proof Files

  • history.json
  • history.json.sig
  • public-key.asc

Verify

gpg --import proofs/public-key.asc
gpg --verify proofs/history.json.sig proofs/history.json
